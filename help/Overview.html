Overview
 
`Penguin Logic' is an application designed to simulate and display the output from a logic circuit. These files are designed to provide the user with information the correct use of the program. N.B. Although this application is tested and should work with Windows, these help files deal with use in Linux only. The following topics are covered:
  • Create a definition file
  • Compile and Run the Application
  • Use the GUI to acheive the desired output
