Compiling and Running the program
    

You will need to compile the application on first use. In the terminal, `cd' to the folder containing the source files (e.g. `Penguin-Logic'), and run the command `make' to compile the program. (N.B. You will need to have wxWidgets and OpenGL libraries installed for the program to compile properly - see separate documentation if there are any issues with this.)

To run the application, run`./logsim grammar_eg1.txt' where`grammar_eg1.txt' is the absolute or relative path of your circuit definition file. Three example definition files have been provided, but to create your own, see here. The definition file will be parsed and if successful, the GUI will be opened. If the parsing is unsuccessful, there is a problem with the definition file and error messages will be output to the terminal. Fix these errors, and try running the program again.
