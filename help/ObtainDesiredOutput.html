Obtain desired output
  
  • Start by clicking the `Run' button. Assuming you have defined some monitors in your definition file, waveforms for those monitors should appear on the screen.
  • The `Continue' button will run the simulation keeping the existing data, whereas the `Run' button will start again. The number of cycles in each simulation can be adjusted with the `Cycles' spinner.
  • To change the device properties, click the `Change device properties' button, and adjust the switch and clock parameters to the desired values.
  • To change the monitor points, click the `Change monitor points' button and select the desired monitors by ticking the appropriate check-boxes.
  • The panel on the right hand side shows the devices entered into the simulator from the circuit definition file.
